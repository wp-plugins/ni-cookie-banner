== Installation ==

1. Upload `ni-cookie-banner` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Set your options from the NI Cookie Banner admin area

Alternatively:

1. Login to your WordPress admin area
2. Search for `NI Cookie Banner`
3. Click install `NI Cookie Banner`
4. Activate through the `Plugins` menu in WordPress or when asked during installation
5. Set your options from the NI Cookie Banner admin area
